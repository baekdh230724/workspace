기본 속성 선택자

    태그에 작성된 특정 속성을 선택하는 선택자
    (모든 속성을 선택할 수 있는 기본 방법)
    (id, class 로 선택 가능하지만 보통은 #, . 을 사용)

    [작성법]
    선택자[속성명="속성값"] { CSS 코드; }

    (참고)
    1) 선택자는 생략할 수 있다
      -> 특정 속성을 가진 모든 요소를 선택

    2) "속성값" 에서 쌍따옴표("")는
     홑따옴표('')로 변경하거나 생략할 수 있다

name 속성 선택하기(name은 html에 기본적으로 존재하는 속성)

div1
div2
div3
div4

data 속성 선택하기(data는 html의 기본속성이 아님)

p1

p2

p3

p4
